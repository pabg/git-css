Comandos

  • git init: crea un repositorio en el directorio actual.
  • mkdir: Se utiliza para crear directorios.
  • rm: este comando sirve para borrar archivos o directorios.
  • touch: Este comando se utiliza para cambiar la fecha de acceso y/o modificacion a un archivo.
  • cat: muestra el contenido de un fichero, por la salida estándar.
  • mv: mueve un fichero o directorio, de un directorio a otro, o le cambia el nombre.
  • cd: cambia el directorio actual.
  • ls: muestra el contenido de un directorio, la lista de ficheros y directorios contenidos en un directorio.
